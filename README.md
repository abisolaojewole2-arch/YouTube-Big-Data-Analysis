# DS203 – YouTube Big Data Analysis Using Apache Spark

## Project Overview
This project analyzes YouTube trending video data across 10 countries using Apache Spark.
The goal is to compare popularity and engagement metrics and uncover regional viewing patterns.

## Dataset
Source: Kaggle – YouTube Trending Videos Dataset
Countries: USA, Great Britain, Germany, Canada, Russia, Mexico, South Korea, Japan, India, France
Records after cleaning: 361,424

## Tools & Technologies
Apache Spark (PySpark)
Spark SQL
Python
Jupyter Notebook
Matplotlib
GitHub

## Key Analysis
Data cleaning and enrichment (country-level tagging)
Filtering, aggregation, grouping, and joins using Spark SQL
Window functions for ranking videos
Engagement rate analysis
Visualization using pie charts and bar charts

## Project Structure
Jupyter notebooks with Spark analysis
Final project report (PDF)
Presentation slides

## How to Run
1. Install Apache Spark and PySpark
2. Open the notebook in Jupyter
3. Update dataset path
4. Run cells sequentially
